# Hotel Booking Project

Un projet de réservation d'hôtel développé avec PHP, JavaScript, Bootstrap, AJAX, SniperJS et CSS. Ce projet permet aux utilisateurs de réserver des chambres d'hôtel en ligne, de consulter la disponibilité des chambres, et d'effectuer des paiements. L'administrateur peut gérer les réservations et les chambres via un panneau d'administration.

## Technologies utilisées

- **PHP** : Langage principal pour le back-end.
- **JavaScript** : Pour les interactions côté client.
- **Bootstrap** : Framework CSS pour une interface responsive et élégante.
- **AJAX** : Pour la gestion des requêtes asynchrones sans recharger la page.
- **SniperJS** : Bibliothèque JavaScript pour améliorer les interactions.
- **CSS** : Pour la personnalisation du design.
  


## Installation

### Prérequis

- PHP version 7.0 ou supérieure
- Serveur Web (Apache ou Nginx recommandé)
- Base de données MySQL

### Étapes d'installation

1. Clonez le repository :
    ```bash
[    git clone https://github.com/basmasaidi00/Hotel-Booking-Project.git
2. Accédez au dossier du projet :
    ```bash
    cd hotel-booking-project
    ```
3. Importez la base de données dans votre gestionnaire de base de données MySQL.
4. Modifiez le fichier de configuration `config.php` pour entrer vos informations de base de données.
5. Lancez votre serveur web et ouvrez le projet dans votre navigateur.
